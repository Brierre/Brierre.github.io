Project One--Rehearsal Scheduler

This is Rehearsal Scheduler V1.0, a tool for musical and play directors to aid in creating a rehearsal calendar. Version 1.0 is a REST application built using Java, MySQL, and Spring Boot. I have big dreams for later versions of this project as I learn more languages and develop my front end skills. For now, it can be cloned and run in your Spring Tool Suite.

Project Two--War

This game doesn't resemble War as I remember it from childhood, but it met the requirements of my boot camp project, and my goal is to recreate it to match more closely with what I recall of the game.

Project Three--Tic-Tac-Toe

This ultra-simplistic Java version of Tic-Tac-Toe is a great starting point for my full-stack training. The plan of my Quick-Projects-for-Learning repository is to find all kinds of short projects useful in learning Java and other languages and showcase them all in one place.

More Projects to Come

It would appear I need more projects to fill up all this space. Fear not, I am working on those.
